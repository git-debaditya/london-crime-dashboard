üìÑ About this Dashboard (Data Processing & Source)

The dataset used in this dashboard was provided as part of the coursework for the module 7CCSMSDV - Introduction to Data Visualisation at King's College London and is originally sourced from Metropolitan Police Service (MPS).

Data cleaning was performed using Python libraries such as pandas and matplotlib. During preprocessing, records containing ambiguous labels such as "Other / NK" were excluded to maintain interpretability.

The data was grouped by offence type and borough, and the total number of offences was calculated to identify trends across different regions of London between February 2021 to January 2025.
